#### Identifying Devices on a Network

- IP Address
- A Media Access Control (MAC) Address

- ##### **<ins>IP Address</ins>** :

it's a way of identifying a host on a network for a period of time

It is composed of a set of numbers divided in four octets

For example the structure is : 

- **192.168.1.1**
- 192 -> Octet #1 (0_255)
- 162 -> Octet #2 (0-255)
- 1 -> Octet #3 (0-255)
- 1 -> Octet #4 (0-255)

> Important to notice that devices can be on both a private and public network

That's why there are 2 types of IP Address :
- Private
- Public

For example :

| Device Name | IP Address | IP Address Type |
|----------------|------------|-----------------|
| DESKTOP-KJE57FD | 192.168.1.77 | 	Private |
| DESKTOP-KJE57FD | 86.157.52.21 | Public |


Public IP Addresses are given by Internet Service Provider (ISP)


Because of the large number of connected devices, all ip adresses has already been adressed, that's why IPV6 was created

- IPV 4 : 2^32 IP ADRESSES
- IPV 6 : 2^128 IP ADRESSES
